My internship @ Nvidia

					Picture with my mentor and team at NVIDIA 
							
 
In the summer of 2024, I interned at NVIDIA as a Hardware ASIC Intern within the CPU verification team, working on UVM-based verification infrastructure for a CHI-based protocol.
The Problem:
	The existing VIP (Verification IP) lacked flag-level control over sub-fields within transactions. Adding this manually — writing the SystemVerilog flag logic, the control code, and the verification tests for each sub-field — was a hours-long process per flag set, and error-prone at scale.
What I Built:
	I developed a Python-based automation tool that took a spec file containing field and sub-field definitions as input and automatically generated three things:
		1. The UVM flag definitions for each sub-field in the transaction
		2. The control logic in SystemVerilog to govern flag behaviour
		3. The back-to-back verification test code to exhaustively validate flag functionality

	The entire workflow — from spec to verified, production-ready SystemVerilog — was fully automated end-to-end.
The Result: 
	The generated flags achieved a 100% pass rate across exhaustive back-to-back testing with zero failures. What previously took hours per flag set was reduced to a script execution. The additions were integrated into the existing VIP and validated within the 2-month internship window.

Thanks to this internship, I gained hands-on experience with code automation and verification. I have also  gained experience working
in an industry setting, collaborating with colleagues, working with tight deadlines, efficient communication and presentation skills.
